    • Yelp Camp

    Full stack Yelp clone that uses RESTful routes to allow adding, editing, and deleting comments and campgrounds. Extra features include a dynamic landing page, passport for authentication and authorization, the Google Maps API to determine and show location for each campground, pagination, and the ability to search through campgrounds.

    Technologies used:

    • Node.js
    • MongoDB
    • Express
    • Mongoose
    • Google Maps API
    • Pagination
    • Passport
    • Database Search
